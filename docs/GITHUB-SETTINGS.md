# GitHub Repository Settings — 配置指南

## 需要手动更新的设置

由于 GitHub 限制，以下设置需要通过 GitHub 网站手动配置：

### 1. About 部分 (仓库首页右侧)

**Description (描述):**
```
Practical reference for agent loop patterns. Stop prompting — design the loop. npm: @kevinzhangnothing/loop-*
```

**Website (网站):**
```
https://KevinZhangNothing.github.io/agent-loops/
```

### 2. Topics (标签)

添加以下 topics 以提高可发现性：

```
agent-loops
ai-agents
coding-agents
claude-code
grok
codex
opencode
cursor
developer-tools
automation
productivity
llm
agentic
context-engineering
```

### 3. 如何更新

#### 添加 Description 和 Website

1. 打开仓库首页：https://github.com/KevinZhangNothing/agent-loops
2. 点击右侧 "About" 部分旁边的 ⚙️ (设置图标)
3. 填写：
   - **Description:** `Practical reference for agent loop patterns. Stop prompting — design the loop. npm: @kevinzhangnothing/loop-*`
   - **Website:** `https://KevinZhangNothing.github.io/agent-loops/`
4. 点击 "Save changes"

#### 添加 Topics

1. 在仓库首页，找到 "About" 部分下方的 "Add topics" 按钮
2. 点击后输入上述 topics（每个 topic 后按回车）
3. 点击 "Save changes"

### 4. 推荐的其他设置

#### Pin Repositories (置顶仓库)

将 agent-loops 置顶到你的 GitHub 个人主页：

1. 访问你的个人主页：https://github.com/KevinZhangNothing
2. 点击 "Customize your pins"
3. 勾选 agent-loops
4. 点击 "Save"

#### Social Preview (社交预览图)

添加社交预览图用于 Twitter/LinkedIn 分享：

1. 进入 Settings → General → Social preview
2. 上传 1280x640 的图片
3. 推荐使用 `assets/visuals/agent-loops-social.png` (待创建)

### 5. 更新后的效果

**仓库首页 About 部分将显示：**
```
📝 Practical reference for agent loop patterns. Stop prompting — design the loop. npm: @kevinzhangnothing/loop-*
🔗 https://KevinZhangNothing.github.io/agent-loops/

Topics: agent-loops · ai-agents · coding-agents · claude-code · grok · codex · opencode · cursor · developer-tools · automation
```

### 6. 为什么需要这些

- **Description:** 让访问者快速了解项目用途
- **Website:** 引导用户到文档和 interactive picker
- **Topics:** 提高 GitHub 搜索排名和可发现性
- **Social Preview:** 在社交媒体分享时显示专业预览图
